# The EBiSC user roles

## user

- no authentication
- can see cell line catalog

## member

- authentication
- can see cell line catalog
- can see executive dashboard

## executive-member

- authentication
- can see cell line catalog
- can see executive dashboard
- can manage executive dashboard

## administrator

- authentication
- can see cell line catalog
- can see executive dashboard
- can manage executive dashboard
- can access admin site
